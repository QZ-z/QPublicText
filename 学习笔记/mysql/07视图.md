# 什么是视图？
- 站在不同的角度去看到数据。（同一张表的数据，通过不同的角度去看待

# 视图的创建与删除
```
create view myview as select empno,ename from emp;

drop view myview;

#注意：只有DQL语句才能以视图对象的方式创建出来。

```
> 对视图进行增删改查，会影响到原表数据。（通过视图影响原表数据的，不是直接操作的原表）
可以对视图进行CRUD操作。

# 视图的作用
- 视图可以隐藏表的实现细节。保密级别较高的系统，数据库只对外提供相关的视图，java程序员
只对视图对象进行CRUD。
